# Azure DevOps Pull Request Review

This extension supports reviewing Azure DevOps pull requests from within VS Code. It is designed around the reviewer workflow rather than pull request authoring.

## Language

**Pull Request Reviewer**:
A user who inspects pull request changes and discussions, communicates feedback, and records a review vote.
_Avoid_: Pull request viewer, pull request author

**Pull Request Authoring**:
The creation and initial configuration of a pull request from a local branch.
_Avoid_: Pull request review, pull request management
